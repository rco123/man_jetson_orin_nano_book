무선 AP 연결

Jetson Nano에서 nmcli를 이용하여 Wi-Fi AP(Access Point)에 접속하는 방법을 단계별로 설명합니다. nmcli는 네트워크 관리 명령줄 도구로, NetworkManager를 통해 네트워크 설정을 수행합니다.

1. Wi-Fi AP에 접속하기 위한 준비

  1. 필수 확인:
  2. Jetson Nano에 Wi-Fi 동글이 장착되어 있는지 확인합니다.

  3. nmcli가 설치되어 있는지 확인합니다 (기본적으로 설치되어 있음).

  4. Wi-Fi AP 정보 확인:

  5. 연결하려는 AP의 SSID(Wi-Fi 이름)와 **비밀번호**를 준비합니다.

2. Wi-Fi 네트워크 검색

  1. Wi-Fi 디바이스 활성화: 
    nmcli radio wifi on
  2. 주변 Wi-Fi 네트워크 검색: 
    nmcli device wifi list
    결과는 다음과 비슷하게 나타납니다: 
    SSID              MODE   CHAN  RATE       SIGNAL  BARS  SECURITY
MyWiFiNetwork     Infra  11    54 Mbit/s  70      ▂▄▆_  WPA2
OtherNetwork      Infra  6     54 Mbit/s  60      ▂▄▆_  WPA2
ai123             Infra  6     54 Mbit/s  60      ▂▄▆_  WPA2

여기서 접속하려는 **SSID**를 확인합니다.

3. Wi-Fi AP에 접속

Wi-Fi 네트워크에 연결하려면 다음 명령어를 실행합니다:

nmcli device wifi connect "<SSID>" password "<비밀번호>"

  • 예: 

    nmcli device wifi connect "ai123" password "1111"

  • 예(비밀번호가없는경우): 

    nmcli device wifi connect "ai123" password ""

4. 연결 상태 확인

Wi-Fi 연결 상태를 확인하려면 다음 명령을 실행합니다:

nmcli connection show

출력 예: 

NAME             UUID                                  TYPE      DEVICE
ai123            12345678-abcd-1234-abcd-1234567890ab  wifi      wlan0

5. 자동 연결 설정

Wi-Fi 네트워크에 자동으로 연결되도록 설정하려면 다음을 실행합니다:

nmcli connection modify "<SSID>" connection.autoconnect yes

6. 네트워크 상태 확인

현재 네트워크 상태와 IP 주소를 확인하려면 다음 명령어를 사용합니다:

nmcli device show wlan0

출력 예: 

GENERAL.DEVICE:                         wlan0
GENERAL.TYPE:                           wifi
GENERAL.STATE:                          100 (connected)
IP4.ADDRESS[1]:                         192.168.1.100/24
IP4.GATEWAY:                            192.168.1.1

7. 문제 해결

7.1 연결 실패 시 로그 확인

journalctl -u NetworkManager

7.2 Wi-Fi 재설정

Wi-Fi 인터페이스를 재설정합니다: 

nmcli device disconnect wlan0
nmcli device connect wlan0

8. 전체 명령 요약

  1. Wi-Fi 스캔: 
    nmcli device wifi list
  2. Wi-Fi 연결: 
    nmcli device wifi connect "<SSID>" password "<비밀번호>"
  3. 연결 상태 확인: 
    nmcli connection show
  4. 자동 연결 설정: 
    nmcli connection modify "<SSID>" connection.autoconnect yes
  5. 네트워크 상태 확인: 
    nmcli device show wlan0

위 과정을 따라 하면 nmcli를 사용하여 Jetson Nano를 Wi-Fi AP에 연결할 수 있습니다. Wi-Fi가 정상적으로 연결되면 인터넷 및 로컬 네트워크를 사용할 수 있습니다.
