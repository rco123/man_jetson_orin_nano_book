Jetson 보드정보

NVIDIA Jetson Orin Nano 개발 보드에서 보드와 리눅스 관련 정보를 확인할 수 있는 명령어를 정리했습니다. 이 명령어들은 보드의 사양, JetPack 버전, 운영체제 및 커널 정보 등을 제공합니다.

Jetson 보드 및 JetPack 정보

Jetson 보드에 대한 구체적인 정보를 확인하려면 아래 명령어를 사용하세요.

1. 보드 정보

sudo tegrastats
  • Jetson 보드의 리소스 사용량(CPU, GPU, RAM, 전력 등)을 실시간으로 모니터링합니다.

2. 보드 모델 및 JetPack 버전

sudo apt-cache show nvidia-jetpack
  • 설치된 JetPack의 버전과 구성 요소를 확인할 수 있습니다.

3. 보드 및 Jetson 정보

jetson_release
  • Jetson의 하드웨어, 소프트웨어, JetPack 버전 등을 상세히 출력합니다.
    • jetson-stats가 설치되어 있어야 합니다. 설치 명령: 
      sudo pip3 install -U jetson-stats

4. 보드 디바이스 트리

cat /proc/device-tree/model
  • Jetson 보드 모델 이름이 출력됩니다.

운영체제 및 커널 정보

1. 운영체제 정보

lsb_release -a
  • 배포판 이름(Ubuntu), 버전 등을 출력합니다.

2. 커널 버전

uname -a
  • 현재 사용 중인 Linux 커널의 버전, 빌드 정보 등을 확인합니다.

3. 호스트 이름

hostname
  • 현재 시스템의 호스트 이름을 확인합니다.

4. 운영체제 및 시스템 정보

cat /etc/os-release
  • 운영체제 이름(Ubuntu), 버전, 코드 이름 등이 포함된 자세한 정보를 제공합니다.

5. CPU 아키텍처

arch
또는 
uname -m

  • CPU 아키텍처(예: aarch64)를 확인합니다.

하드웨어 관련 정보

1. CPU 정보

lscpu
  • CPU의 상세 사양(코어 수, 클럭 속도 등)을 확인합니다.

2. 메모리 정보

free -h
  • 시스템의 총 메모리, 사용 중인 메모리, 여유 메모리 등을 확인합니다.

3. 스토리지 정보

df -h
  • 파일 시스템의 사용량과 가용 공간을 확인합니다.

4. I/O 포트 정보

lsblk
  • 연결된 저장 장치 정보를 확인합니다.

네트워크 관련 정보

1. IP 주소 확인

ip addr
또는 
ifconfig

2. 네트워크 설정

nmcli device show
  • 네트워크 인터페이스 및 설정 정보를 출력합니다.

Jetson에 특화된 도구

1. NVIDIA 클럭 속도 확인

sudo jetson_clocks --show
  • Jetson의 클럭 속도를 확인합니다.

2. Jetson 클럭 최적화

sudo jetson_clocks
  • Jetson의 CPU/GPU 클럭 속도를 최대화합니다.

3. 전력 모드 설정

sudo nvpmodel -q
  • 현재 전력 모드를 확인합니다.
sudo nvpmodel -m <모드 번호>
  • 전력 모드를 변경합니다. 모드 번호는 보드 사양에 따라 다릅니다(예: 최대 성능 모드, 저전력 모드).
