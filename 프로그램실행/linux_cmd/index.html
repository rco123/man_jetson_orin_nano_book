쉘명령어

Jetson Nano 보드를 사용하는데, 자주 사용되는 리눅스 기본 명령어를 이해하고 사용할 수 있다면, 리눅스를 다루는 데 큰 도움이 됩니다.

아래는 초보자들이 알아야 할 주요 리눅스 명령어를 간단히 정리한 목록입니다.

파일 및 디렉토리 관리

  1. pwd (Print Working Directory)
    현재 작업 중인 디렉토리의 경로를 표시합니다. 

    pwd

  2. ls (List)
    현재 디렉토리의 파일 및 디렉토리 목록을 표시합니다. 

    ls
ls -l    # 자세한 정보 표시
ls -a    # 숨김 파일 포함

  3. cd (Change Directory)
    디렉토리를 변경합니다. 

    cd /path/to/directory  # 특정 경로로 이동
cd ..                  # 상위 디렉토리로 이동
cd ~                   # 홈 디렉토리로 이동

  4. mkdir (Make Directory)
    새 디렉토리를 생성합니다. 

    mkdir new_folder

  5. rm (Remove)
    파일 또는 디렉토리를 삭제합니다. 

    rm file_name          # 파일 삭제
rm -r folder_name     # 디렉토리와 내용 삭제

  6. cp (Copy)
    파일 또는 디렉토리를 복사합니다. 

    cp source_file target_file
cp -r source_folder target_folder

  7. mv (Move)
    파일 또는 디렉토리를 이동하거나 이름을 변경합니다. 

    mv old_name new_name
mv file_name /path/to/target_directory

파일 보기 및 편집

  1. cat (Concatenate)
    파일 내용을 표시하거나 파일을 연결합니다. 

    cat file_name

  2. less / more
    파일 내용을 페이지 단위로 확인합니다. 

    less file_name
more file_name

  3. nano / vi / vim
    파일을 편집합니다. (초보자는 nano를 추천) 

    nano file_name
vi file_name
vim file_name

  4. touch
    새 파일을 생성하거나 기존 파일의 날짜/시간을 업데이트합니다. 

    touch new_file

시스템 정보

  1. whoami
    현재 사용자의 이름을 출력합니다. 

    whoami

  2. uname
    시스템 정보를 표시합니다. 

    uname
uname -a  # 자세한 정보

  3. df (Disk Free)
    디스크 공간을 확인합니다. 

    df -h

  4. du (Disk Usage)
    디렉토리 또는 파일의 크기를 확인합니다. 

    du -h
du -sh folder_name

  5. top / htop
    현재 실행 중인 프로세스와 시스템 상태를 표시합니다. 

    top
htop  # (htop은 설치 필요)

권한 및 소유권 관리

  1. chmod (Change Mode)
    파일의 권한을 변경합니다. 

    chmod 755 file_name  # 읽기/쓰기/실행 권한

  2. chown (Change Ownership)
    파일 또는 디렉토리의 소유자를 변경합니다. 

    chown user:group file_name

네트워크

  1. ping
    네트워크 연결 상태를 확인합니다. 

    ping google.com

  2. curl / wget
    파일을 다운로드하거나 URL 요청을 보냅니다. 

    curl https://example.com
wget https://example.com/file.zip

기타 유용한 명령어

  1. history
    사용자가 실행한 명령어 목록을 확인합니다. 

    history

  2. clear
    터미널 화면을 정리합니다. 

    clear

  3. man (Manual)
    명령어의 매뉴얼을 표시합니다. 

    man ls

  4. exit
    터미널 세션을 종료합니다. 

    exit
