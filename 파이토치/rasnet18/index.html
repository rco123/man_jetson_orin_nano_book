RasNet18 구현

아래는 PyTorch를 사용하여 ResNet18을 활용한 전이 학습 과정을 구현한 예제입니다. 주어진 디렉토리에 있는 이미지를 로드하고, 평균과 표준편차를 계산하며, 전처리 및 전이 학습을 수행한 후 이미지를 -45도 ~ +45도 각도로 예측하는 코드입니다.

주요 단계:

  1. 데이터 로드 및 평균/표준편차 계산.
  2. 이미지 전처리 (ResNet18 요구 사항에 맞게).
  3. ResNet18 모델을 활용한 전이 학습 설정.
  4. 모델 학습 및 테스트 데이터로 결과 출력.

코드:

import os
import torch
import torch.nn as nn
import torch.optim as optim
from torchvision import datasets, transforms, models
from torch.utils.data import DataLoader
import numpy as np
import matplotlib.pyplot as plt

# 1. 데이터 로드 및 평균/표준편차 계산
def compute_mean_std(data_loader):
    mean = 0.0
    std = 0.0
    total_images = 0

    for images, _ in data_loader:
        batch_samples = images.size(0)  # 배치 크기
        images = images.view(batch_samples, images.size(1), -1)  # (N, C, H*W)
        mean += images.mean(2).sum(0)
        std += images.std(2).sum(0)
        total_images += batch_samples

    mean /= total_images
    std /= total_images
    return mean, std


# 2. 데이터셋 정의 및 로드
data_dir = "path_to_Rasnet18_directory"  # 데이터셋 경로
transform = transforms.Compose([
    transforms.Resize((224, 224)),  # ResNet18의 입력 크기에 맞춤
    transforms.ToTensor()  # Tensor로 변환
])

dataset = datasets.ImageFolder(data_dir, transform=transform)
data_loader = DataLoader(dataset, batch_size=32, shuffle=True)

# 평균과 표준편차 계산
mean, std = compute_mean_std(data_loader)
print(f"Mean: {mean}, Std: {std}")

# 평균과 표준편차를 사용하여 데이터 정규화
transform_normalized = transforms.Compose([
    transforms.Resize((224, 224)),
    transforms.ToTensor(),
    transforms.Normalize(mean, std)
])

# 데이터 로더 업데이트
dataset = datasets.ImageFolder(data_dir, transform=transform_normalized)
data_loader = DataLoader(dataset, batch_size=32, shuffle=True)

# 3. ResNet18 모델 로드 및 수정
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = models.resnet18(pretrained=True)

# 기존 마지막 레이어를 각도 회귀를 위한 출력 레이어로 수정
model.fc = nn.Linear(model.fc.in_features, 1)
model = model.to(device)

# 4. 손실 함수 및 옵티마이저 설정
criterion = nn.MSELoss()  # 회귀 문제에 적합한 손실 함수
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 5. 모델 학습
def train_model(model, data_loader, criterion, optimizer, num_epochs=5):
    model.train()
    for epoch in range(num_epochs):
        running_loss = 0.0
        for images, labels in data_loader:
            # 라벨을 -45도 ~ 45도 범위로 매핑 (임의로 설정)
            angles = torch.rand(len(labels)) * 90 - 45
            angles = angles.view(-1, 1).to(device)  # 회귀값에 맞게 reshape

            images = images.to(device)

            # Forward
            outputs = model(images)
            loss = criterion(outputs, angles)

            # Backward
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

            running_loss += loss.item()

        print(f"Epoch [{epoch + 1}/{num_epochs}], Loss: {running_loss / len(data_loader):.4f}")


# 학습 수행
train_model(model, data_loader, criterion, optimizer, num_epochs=5)

# 6. 결과 확인: -45도 ~ 45도 각도 출력
def test_model(model, data_loader):
    model.eval()
    with torch.no_grad():
        for images, _ in data_loader:
            images = images.to(device)
            outputs = model(images).cpu().numpy()

            for i, output in enumerate(outputs):
                angle = output[0]
                print(f"Predicted Angle: {angle:.2f} degrees")

                # 첫 번째 이미지를 예제로 출력
                if i == 0:
                    img = images[i].cpu().numpy().transpose((1, 2, 0)) * std.numpy() + mean.numpy()
                    img = np.clip(img, 0, 1)
                    plt.imshow(img)
                    plt.title(f"Predicted Angle: {angle:.2f}°")
                    plt.axis("off")
                    plt.show()
                    break


# 테스트 실행
test_model(model, data_loader)

코드 설명:

  1. 데이터 로드 및 전처리:
  2. datasets.ImageFolder를 사용해 데이터를 로드하며, ResNet18의 입력 요구사항에 맞게 크기를 조정합니다.

  3. 데이터의 평균(mean)과 표준편차(std)를 계산하고, 이를 사용해 데이터를 정규화합니다.

  4. ResNet18 수정:

  5. 사전 학습된 ResNet18 모델을 불러온 뒤 마지막 Fully Connected Layer를 1개 출력값으로 수정합니다. 이는 각도를 예측하기 위한 회귀 문제에 맞춘 설정입니다.

  6. 학습:

  7. 랜덤하게 생성된 각도를 정답값으로 설정(-45도 ~ +45도).

  8. MSELoss를 손실 함수로 사용하여 학습합니다.

  9. 테스트 및 출력:

  10. 학습된 모델을 사용하여 각도를 예측하고, 이미지를 출력하며 예측 각도를 표시합니다.
