opencv 이미지 데이터 변환

Python의 Pillow 라이브러리(기본 Image 모듈)를 사용하여 OpenCV, Matplotlib, Pillow 간의 이미지 데이터를 상호 변환하고 출력하는 방법을 추가로 설명하겠습니다.

1. Pillow 라이브러리 기본 소개

Pillow는 이미지 처리와 조작을 위한 Python 라이브러리입니다. Pillow는 다양한 이미지 형식(BMP, PNG, JPEG 등)을 지원하며, OpenCV 및 Matplotlib과도 데이터를 상호 변환할 수 있습니다.

Pillow 설치

pip install pillow

2. OpenCV ↔ Pillow 데이터 변환

OpenCV 이미지 → Pillow 이미지 변환

OpenCV는 NumPy 배열 형식의 이미지를 사용합니다. 이를 Pillow로 변환하려면 Image.fromarray를 사용합니다.

코드:

import cv2
from PIL import Image

# OpenCV로 이미지 읽기 (BGR)
image = cv2.imread("sample.jpg")

# OpenCV (BGR) → Pillow (RGB)
image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)  # BGR → RGB 변환
image_pillow = Image.fromarray(image_rgb)  # NumPy 배열 → Pillow 이미지
image_pillow.show()  # Pillow로 이미지 출력

Pillow 이미지 → OpenCV 이미지 변환

Pillow 이미지를 OpenCV 형식으로 변환하려면 np.array를 사용합니다.

코드:

import cv2
from PIL import Image
import numpy as np

# Pillow로 이미지 열기
image_pillow = Image.open("sample.jpg")

# Pillow → OpenCV
image_rgb = np.array(image_pillow)  # Pillow → NumPy 배열
image_bgr = cv2.cvtColor(image_rgb, cv2.COLOR_RGB2BGR)  # RGB → BGR 변환
cv2.imshow("OpenCV Image", image_bgr)  # OpenCV로 출력
cv2.waitKey(0)
cv2.destroyAllWindows()

3. Matplotlib ↔ Pillow 데이터 변환

Matplotlib는 Pillow와 동일하게 RGB 형식을 사용하므로 변환이 간단합니다.

Pillow 이미지 → Matplotlib

Pillow 이미지를 Matplotlib로 변환하여 출력하려면 np.array를 사용합니다.

코드:

from PIL import Image
import matplotlib.pyplot as plt
import numpy as np

# Pillow로 이미지 열기
image_pillow = Image.open("sample.jpg")

# Pillow → Matplotlib (NumPy 배열로 변환)
image_rgb = np.array(image_pillow)

# Matplotlib로 출력
plt.imshow(image_rgb)
plt.title("Image with Matplotlib")
plt.axis("off")
plt.show()

Matplotlib 이미지 → Pillow

Matplotlib는 NumPy 배열 형식을 사용합니다. 이를 Pillow 형식으로 변환하려면 Image.fromarray를 사용합니다.

코드:

from PIL import Image
import numpy as np

# 가상 NumPy 배열 생성 (Matplotlib에서 처리된 데이터라고 가정)
image_rgb = np.zeros((300, 300, 3), dtype=np.uint8)
image_rgb[:, :150] = [255, 0, 0]  # 왼쪽은 빨간색
image_rgb[:, 150:] = [0, 255, 0]  # 오른쪽은 초록색

# Matplotlib (NumPy 배열) → Pillow
image_pillow = Image.fromarray(image_rgb)
image_pillow.show()  # Pillow로 출력

4. OpenCV ↔ Matplotlib ↔ Pillow 간 데이터 흐름

OpenCV → Pillow → Matplotlib

import cv2
from PIL import Image
import matplotlib.pyplot as plt
import numpy as np

# OpenCV로 이미지 읽기 (BGR)
image = cv2.imread("sample.jpg")

# OpenCV (BGR) → Pillow (RGB)
image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
image_pillow = Image.fromarray(image_rgb)

# Pillow → Matplotlib
image_rgb_array = np.array(image_pillow)

# Matplotlib로 출력
plt.imshow(image_rgb_array)
plt.title("Image with Matplotlib")
plt.axis("off")
plt.show()

데이터 흐름 요약

라이브러리 입력 형식 출력 형식 변환 메서드
OpenCV → Pillow NumPy 배열 (BGR) Pillow 이미지 (RGB) cv2.cvtColor + Image.fromarray
Pillow → OpenCV Pillow 이미지 (RGB) NumPy 배열 (BGR) np.array + cv2.cvtColor
Matplotlib → Pillow NumPy 배열 (RGB) Pillow 이미지 (RGB) Image.fromarray
Pillow → Matplotlib Pillow 이미지 (RGB) NumPy 배열 (RGB) np.array

5. 각 라이브러리의 이미지 출력 차이

라이브러리 출력 방식 장점 제한사항
OpenCV 독립 GUI 창에서 출력 실시간 처리, 이벤트 기반 처리 가능 BGR 형식 사용, Jupyter에 부적합
Matplotlib 그래프 내에서 시각화 출력 데이터 분석 및 시각화에 유리 실시간 처리에 부적합
Pillow OS 기본 이미지 뷰어에서 출력 이미지 변환 및 파일 작업에 유리 복잡한 시각화는 제한적

요약

  • OpenCV, Matplotlib, Pillow는 각기 다른 강점을 가진 이미지 처리 라이브러리입니다.
  • 데이터를 상호 변환하면 각 라이브러리의 장점을 조합하여 활용할 수 있습니다.
  • Pillow는 OpenCV와 Matplotlib 간의 데이터 변환에 중요한 역할을 합니다.
