opencv 이미지출력

이미지를 출력하는 방법은 OpenCV와 Matplotlib를 사용하는 두 가지 주요 방식이 있습니다. 두 라이브러리는 서로 다른 방식으로 이미지를 처리하며, 상호 데이터 변환을 통해 각 라이브러리의 장점을 활용할 수 있습니다.

1. OpenCV로 이미지 출력

OpenCV의 imshow 함수는 이미지를 화면에 직접 출력하는 기능을 제공합니다.

코드:

import cv2

# 이미지 읽기
image = cv2.imread("sample.jpg")  # OpenCV는 이미지를 기본적으로 BGR 형식으로 읽음.

# OpenCV로 이미지 출력
cv2.imshow("OpenCV Image", image)

# 키 입력 대기
cv2.waitKey(0)  # 0이면 무한 대기, 특정 시간(ms)을 입력하면 그 시간만큼 대기.
cv2.destroyAllWindows()

특징:

  • BGR 포맷 사용: OpenCV는 이미지를 BGR 형식으로 처리합니다.
  • 실시간 출력: imshow는 창을 열어 실시간으로 이미지를 출력합니다.
  • GUI 창 사용: 독립적인 GUI 창에서 이미지를 보여줍니다.

2. Matplotlib로 이미지 출력

Matplotlib의 plt.imshow 함수는 이미지를 그래프로 출력하는 데 사용됩니다. Matplotlib는 이미지를 기본적으로 RGB 형식으로 처리합니다.

코드:

import cv2
import matplotlib.pyplot as plt

# 이미지 읽기
image = cv2.imread("sample.jpg")  # OpenCV는 BGR 형식으로 이미지를 읽음.

# BGR을 RGB로 변환 (Matplotlib는 RGB 형식으로 처리)
image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)

# Matplotlib로 이미지 출력
plt.imshow(image_rgb)
plt.title("Matplotlib Image")
plt.axis("off")  # 축 제거
plt.show()

특징:

  • RGB 포맷 사용: Matplotlib는 이미지를 RGB 형식으로 처리하므로 변환이 필요합니다.
  • 통합 환경에서 출력: Jupyter Notebook 등에서 시각화에 유리합니다.
  • 축 옵션: 축 제거, 라벨링 등 그래프 기능을 제공합니다.

3. OpenCV와 Matplotlib 간 데이터 변환

OpenCV와 Matplotlib는 서로 다른 색상 형식을 사용하기 때문에, 이미지를 출력할 때 적절한 색상 변환이 필요합니다.

변환 방법:

  1. OpenCV (BGR) → Matplotlib (RGB): 

    image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)

  2. Matplotlib (RGB) → OpenCV (BGR): 

    image_bgr = cv2.cvtColor(image, cv2.COLOR_RGB2BGR)

4. 상호 출력 예제

OpenCV로 이미지를 읽고 Matplotlib로 출력

import cv2
import matplotlib.pyplot as plt

# OpenCV로 이미지 읽기
image = cv2.imread("sample.jpg")  # BGR 형식

# BGR → RGB 변환
image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)

# Matplotlib로 출력
plt.imshow(image_rgb)
plt.title("Image in RGB (Matplotlib)")
plt.axis("off")
plt.show()

Matplotlib로 이미지를 처리하고 OpenCV로 출력

import cv2
import matplotlib.pyplot as plt
import numpy as np

# 가상 이미지 생성 (Matplotlib에서 처리)
image_rgb = np.zeros((300, 300, 3), dtype=np.uint8)
image_rgb[:, :150] = [255, 0, 0]  # 왼쪽은 빨간색
image_rgb[:, 150:] = [0, 255, 0]  # 오른쪽은 초록색

# RGB → BGR 변환
image_bgr = cv2.cvtColor(image_rgb, cv2.COLOR_RGB2BGR)

# OpenCV로 출력
cv2.imshow("Image in BGR (OpenCV)", image_bgr)
cv2.waitKey(0)
cv2.destroyAllWindows()

5. OpenCV와 Matplotlib의 출력 차이

특징 OpenCV Matplotlib
색상 포맷 BGR RGB
창 관리 독립적인 GUI 창 Jupyter Notebook, 통합 환경 적합
추가 기능 실시간 처리, 키 입력 이벤트 그래프 기능, 축 표시 및 제거 가능
사용 사례 영상 처리, 실시간 애플리케이션 데이터 분석 및 시각화

요약

  • OpenCV는 실시간 처리 및 키 입력 대기 기능에 적합합니다.
  • Matplotlib는 데이터 시각화 및 분석에 적합합니다.
  • 두 라이브러리는 색상 포맷이 다르므로 변환(cv2.cvtColor)이 필요합니다.
